# ImgurViewer
Android application that retrieves images from the Imgur server to create an image gallery.

## Dependencies
ImgurViewer retrieves content from Imgur's [API](https://api.imgur.com/)

Sign up with Imgur as a developer and edit Common.java file with your own client id:
```
IMGUR_CLIENT_ID=your_client_id
```

## Libraries
* [Volley](https://github.com/mcxiaoke/android-volley)
* [Picasso](http://square.github.io/picasso/)
